Cell


Cell is my video game engine project that I've been working on since about 2019.

I started with no experience at all, and had to restart several times.


The main goal is to make a lightweight and easily usable engine. It is currently written in C++, and I'm considering porting some components to Rust with a thin wrapper.

At the minute, it features a completely custom Vulkan renderer, an SSE accelerated mathematics module for linear algebra, a pretty crooked JSON and glTF parser, as well as broken module implementations for WASAPI/PulseAudio and OpenXR.
